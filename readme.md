# Reception Dashboard

A simple reception management dashboard built as a school project. The application simulates a front-desk interface where staff presence and deliveries can be tracked in a clear, visual layout.

## Features
- Staff in/out tracking
- Delivery scheduling interface
- Live digital clock
- Toast notifications for actions
- Dashboard-style layout

## Tech Stack
- HTML
- CSS
- JavaScript
- Bootstrap

## Project Purpose
This project was created during my first semester at Noroff.  
The goal was to practice:

- DOM manipulation
- Event handling
- UI state updates
- Building a dashboard-style interface
- Structuring front-end logic

## How to Run
1. Clone the repository
2. Open the project folder
3. Open index.html in your browser

No build tools or server required.

## Notes
This is a front-end learning project focused on interface logic and user interaction rather than backend functionality.
